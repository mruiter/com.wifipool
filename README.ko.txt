베니페로 WiFi 수영장

Beniferro의 WiFi 수영장 제어

레독스, pH, 온도 및 유량 모니터링 지원 추가

앱 설정 페이지: 로그인 테스트/로그, 기기 도메인 및 연결된 모듈 찾기.
기기를 추가할 때 인증 정보와 검색이 기기에 대해 수행됩니다.
기기: 하나의 "WiFi 수영장 센서"가 `measure_redox`, `measure_ph`, `measure_temperature`, `alarm_flow` 기능을 제공합니다.
로그: `homey app run`을 실행할 때 광범위한 로그가 출력됩니다.

## 주의 사항
API 엔드포인트는 통신 캡처를 기반으로 하며, 공식 문서는 제공되지 않습니다.
중요: Beniferro 측의 변경 사항은 통보되지 않으며 이 앱의 기능을 중단시킬 수 있습니다.
