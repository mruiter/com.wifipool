# Beniferro WiFi Pool

Контроль WiFi бассейна от Beniferro

Добавьте поддержку мониторинга вашего Redox, pH, температуры и расхода

- Страница настроек приложения: Просто для тестирования/логирования входа в систему, поиска домена устройства и подключенных модулей.
- При добавлении устройства учетные данные и обнаружение будут выполнены для устройства.
- Устройство: один "WiFi Pool Sensor" раскрывает возможности `measure_redox`, `measure_ph`, `measure_temperature`, `alarm_flow`.
- Логирование: подробные логи печатаются при выполнении команды `homey app run`.

## Примечания
- Конечные точки API основаны на захвате коммуникации, официальная документация недоступна.
  - Важно: Любые изменения на стороне Beniferro не сообщаются и могут нарушить работоспособность этого приложения.
