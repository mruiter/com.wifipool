# 베니페로 WiFi 수영장

베니페로 WiFi 수영장 제어

당신의 레독스, pH, 온도 및 흐름을 모니터링할 수 있도록 지원 추가

- 앱 설정 페이지: 로그인 테스트/로그 기록, 장치 도메인 및 연결된 모듈 찾기.
- 장치 추가 시 자격 증명 및 디스커버리가 장치에 대해 실행됩니다.
- 장치: 하나의 "WiFi 수영장 센서"는 `measure_redox`, `measure_ph`, `measure_temperature`, `alarm_flow` 기능을 노출합니다.
- 로깅: `homey app run` 실행 시 광범위한 로그 출력.

## 주의 사항
- API 엔드포인트는 통신 캡처를 기반으로 하며, 공식 문서는 제공되지 않습니다.
  - 중요: 베니페로 측의 변경 사항은 통보되지 않으며 이 앱의 기능을 무너트릴 수 있습니다.
