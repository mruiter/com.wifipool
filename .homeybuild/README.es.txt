Beniferro WiFi Pool

Control de piscina WiFi de Beniferro

Añade soporte para monitorear Redox, pH, temperatura y flujo

Página de configuración de la app: Solo para probar y registrar el inicio de sesión, encontrar el dominio del dispositivo y los módulos conectados. Al agregar un dispositivo, las credenciales y el descubrimiento se realizarán para el dispositivo.
Dispositivo: un "Sensor de piscina WiFi" expone las capacidades `measure_redox`, `measure_ph`, `measure_temperature`, `alarm_flow`.
Registro: se imprimen extensos registros al ejecutar `homey app run`.

## Notas
Los puntos de conexión API se basan en capturar la comunicación, la documentación oficial no está disponible.
Importante: Cualquier cambio del lado de Beniferro no se comunica y podría romper la funcionalidad de esta aplicación.
