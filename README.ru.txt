Beniferro WiFi Pool

Управление WiFi Pool от Beniferro

Добавьте поддержку для мониторинга Redox, pH, температуры и потока

Страница настроек приложения: просто для тестирования/регистрации логина, поиска домена устройства и подключенных модулей. При добавлении устройства будут выполнены ввод данных и обнаружение устройства. Устройство: один "WiFi Pool Sensor" обладает возможностями `measure_redox`, `measure_ph`, `measure_temperature`, `alarm_flow`. Логирование: обширные логи выводятся при запуске `homey app run`.

## Примечания
Точки доступа API основаны на перехвате связи, официальная документация недоступна. Важно: любые изменения со стороны Beniferro не сообщаются и могут нарушить работоспособность этого приложения.
